# Elasticsearch Expert Training (2-Day Course)

Welcome to the comprehensive 2-day Elasticsearch training program. This repository contains all theory modules, architecture diagrams, and hands-on lab instructions tailored for Ubuntu environments.

## Course Structure (8 Hours / Day)

### Day 1: Foundations, Setup, and Data Ingestion
1. **[Module 1: Introduction, Architecture & Foundations](module-1.md)**
   - [Lab 1: JSON & REST API Basics](lab-1.md)
   - [Lab 2: Architecture Whiteboarding](lab-2.md)
2. **[Module 2: Installation, Security & High Availability](module-2.md)**
   - [Lab 3: Installing Elasticsearch & Kibana on Ubuntu](lab-3.md)
   - [Lab 4: Configuring Basic Security](lab-4.md)
3. **[Module 3: Data Ingestion & Processing](module-3.md)**
   - [Lab 5: The Bulk Indexing API](lab-5.md)
   - [Lab 6: Installing and Configuring Filebeat](lab-6.md)
   - [Lab 7: Explicit Mappings & Ingest Pipelines](lab-7.md)

### Day 2: Search, Data Lifecycle, and Advanced APIs
4. **[Module 4: Search, Query DSL & Relevance](module-4.md)**
   - [Lab 8: Query vs. Filter Contexts](lab-8.md)
   - [Lab 9: Aggregations Framework](lab-9.md)
   - [Lab 10: Analyzing Search Performance](lab-10.md)
5. **[Module 5: Data Lifecycle Management & Optimization](module-5.md)**
   - [Lab 11: Implementing Index Lifecycle Management (ILM)](lab-11.md)
   - [Lab 12: Configuring Local Snapshots](lab-12.md)
6. **[Module 6: APIs, Clients & Advanced Topics](module-6.md)**
   - [Lab 13: Using Painless Scripts](lab-13.md)
   - [Lab 14: Simulating & Troubleshooting Cluster Issues](lab-14.md)

## Instructor Preparation
Please ensure the students' Ubuntu environments have `curl`, `wget`, and standard developer packages installed prior to `Day 1 - Hour 3`.
