# Elasticsearch Expert Training (2-Day Course)

Welcome to the comprehensive 2-day Elasticsearch training program. This repository contains all theory modules, architecture diagrams, and hands-on lab instructions tailored for Ubuntu environments.

### Day 1: Foundations, Setup, and Data Ingestion
1. **[Module 1: Architecture, Indices, Shards](module1/module1.md)**

2. **[Module 2: HA Clusters, Troubleshooting, Upgrading](module2/module2.md)**
   - [Lab 3: Installing Elasticsearch & Kibana on Ubuntu](module2/lab3.md)
   - [Lab 4: Configuring Basic Security & Kibana Setup](module2/lab4.md)
   - [Lab 5: Simulating & Advanced Troubleshooting](module2/lab5.md)
3. **[Module 3: Data Ingestion, Logstash, Beats](module3/module3.md)**
   - [Lab 6: Indexing Data via Bulk API](module3/lab6.md)
   - [Lab 7: Installing Filebeat & Logstash](module3/lab7.md)
   - [Lab 8: Explicit Mappings & Ingest Pipelines](module3/lab8.md)

### Day 2: Search, Data Lifecycle, and Advanced APIs
4. **[Module 4: Search DSL, ES|QL, EQL, SQL](module4/module4.md)**
   - [Lab 9: Query vs. Filter Contexts](module4/lab9.md)

Please ensure the students' Ubuntu environments have `curl`, `wget`, `docker.io`, and standard developer packages installed prior to Day 1.
