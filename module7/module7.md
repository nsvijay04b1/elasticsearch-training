# Module 7: Advanced Use Cases & Day 2 Operations

## Overview
This module explores expert-level Elasticsearch use cases, taking you beyond basic ingestion and search. We cover security threat hunting, active application debugging, and "Day 2" operations — the critical tasks required to keep a production cluster healthy, performant, and resilient.

## Topics Covered
| Lab | Topic | Description | Level |
|---|---|---|---|
| Lab 22 | Security Threat Hunting | Ingest dummy auth logs, use ES\|QL to detect brute force attacks and visualize anomalies | Advanced |
| Lab 23 | Application Debugging | Correlate application logs with APM traces, finding slow transactions and unhandled exceptions | Advanced |
| Lab 24 | Performance Troubleshooting | Use `_nodes/hot_threads` and Profile API to debug slow queries and high CPU (Day 2) | Advanced |
| Lab 25 | Advanced Data Lifecycle | Create a robust ILM policy covering Hot, Warm, Cold, and Delete phases (Day 2) | Intermediate |
| Lab 26 | Cluster Resilience | Diagnose complex shard allocation failures and perform zero-downtime rolling restarts (Day 2) | Advanced |
